О системе управления автопарком

Возможности системы

Система управления автопарком предоставляет удобные инструменты для контроля и учета автотранспорта вашей организации.

Веб-интерфейс
  • Просмотр всех транспортных средств
  • Детальная информация о каждом ТС
  • История технического обслуживания
  • Учет ремонтов и их стоимости
  • Статистика по расходу топлива
  • Контроль сроков страховки и ТО
Telegram Бот
  • Оперативный доступ к данным
  • Уведомления о предстоящем ТО
  • Внесение данных о заправках
  • Регистрация ремонтов
  • Обновление пробега ТС
  • Администрирование системы

Особые функции

Контроль документов

Система отслеживает сроки действия ОСАГО, технического осмотра и тахографа

Учет расходов

Полный учет затрат на топливо и ремонты с формированием отчетов

Уведомления

Автоматические оповещения о приближающихся сроках обслуживания

Техническая информация

Система разработана с использованием современных технологий:

  • Серверная часть: Python (Flask)
  • База данных: SQLite
  • Интерфейс пользователя: HTML5, CSS3, JavaScript
  • Telegram Bot API для мобильного доступа

Для получения доступа к Telegram боту, перейдите по ссылке @check_vin_avtobot и нажмите кнопку "Запустить".
